# XHS Seeding Layouts

Build Xiaohongshu 3:4 carousels that are useful enough to save and specific enough to convert.

## Default Merchant Carousel

1. Cover: concrete scene promise, not a vague slogan.
2. Why now: summer, holiday, weekend, exhibition period, limited workshop, seasonal product.
3. What you get: route, experience, product, service, or finished work.
4. Detail proof: craft process, exhibit detail, menu, makeup, room, route map, real photo cue.
5. Suitable for: parent-child, friends, couples, solo, tourists, locals, beginners.
6. Practical info: time, reservation, dress, transportation, price range if provided.
7. CTA: save, comment keyword, book, visit, ask for route, or reserve.

Use 5 pages for simple offers:

1. Cover
2. Why worth going
3. Experience or route
4. Photo/material highlights
5. CTA and practical notes

## Cover Title Formulas

- 暑假带娃来这里, 不只是打卡
- 这条国风路线, 适合慢慢逛
- 夏天想避暑, 可以来这里坐半天
- 一次能拍、能学、能带走的非遗体验
- 不想拍游客照, 这几个点更出片
- 新中式下午茶, 适合约朋友慢坐

## Copy Rules

- Lead with a concrete user scenario.
- Use saveable information: route, checklist, process, photo spots, suitable people, booking reminders.
- Keep each page short: one title and 2-4 lines.
- Mention conversion information only if the user provides it.
- Replace hard selling with practical next steps.

## Material Usage Advice

- Entrance or sign: use for trust and location context.
- Product close-up: use for cover if visual identity is strong.
- Process photo: use for non-heritage and workshop credibility.
- Customer sample: use for hanfu, craft, parent-child, or finished-product pages.
- Space photo: use for tea, guesthouse, garden, and travel contexts.
- Map or route screenshot: turn into a saveable itinerary page.
